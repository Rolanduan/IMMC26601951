\section*{AI Use Report}
\addcontentsline{toc}{section}{AI Use Report}
This report follows current AI-enabled research practice (as of 2026), where large models are strong copilots for synthesis, coding, and drafting, but not autonomous decision-makers for scientific claims.

In this work, AI was used to:
\begin{itemize}
  \item co-design paper structure and section logic aligned with IMMC requirements;
  \item assist symbolic formulation, algorithm pseudocode drafting, and LaTeX expression polishing;
  \item generate and refine figure scripts (SVG/Python-style plotting workflow);
  \item accelerate language editing for clarity, concision, and technical consistency.
\end{itemize}

Our technical view of modern AI is pragmatic:
\begin{itemize}
  \item \textbf{Multimodal and agentic AI} is now effective for combining text, code, and diagram generation in one pipeline.
  \item \textbf{Retrieval/tool-augmented workflows} improve productivity but do not remove the need for domain validation.
  \item \textbf{Probabilistic generation risk} (hallucination, overconfident wording, hidden inconsistency) remains real, especially for numerical and causal claims.
\end{itemize}

Therefore, all final model assumptions, equations, parameter interpretations, and conclusions were human-reviewed. We applied explicit safeguards: dimensional checks, constraint-feasibility checks, sensitivity sanity checks, and consistency checks between text, equations, code, and figures. AI outputs were treated as draft material; responsibility for the final scientific content remains with the authors.
